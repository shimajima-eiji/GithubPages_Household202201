UbuntuのLiveCD(GUI)でいうところの、
- インストーラDVDで起動
- 「インストールせずにUbuntuを試す」を選択
- 「terminal(端末)」 or cmdを開く
  - Ctrl+Alt+F2でコンソールを開いた場合、ubuntu login:で ubuntu/(Enter)でログイン
- 元のディスクをマウントしてchrootする
でしょうか。
